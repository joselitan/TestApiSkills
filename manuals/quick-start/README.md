# Quick Start Guide

Detta är den viktigaste manualen - första intryck avgör allt!

## 📁 Filer

- `quick-start-guide.md` - Markdown-version (källfil)
- `quick-start-guide.html` - HTML-version (professionell presentation)
- `quick-start-guide.pdf` - PDF-version (genereras med script)

## 🎯 Syfte

Få användare att komma igång på **30 minuter** med:
- Installation och setup
- Första test-körning
- Dashboard overview
- Grundläggande CI/CD

## 📊 Målgrupp

- **Primär:** QA-studenter som är nya på plattformen
- **Sekundär:** Instruktörer som ska demo:a
- **Tertiär:** Potentiella kunder som utvärderar

## ✅ Kvalitetskrav

- [ ] Alla kommandon testade och fungerar
- [ ] Screenshots aktuella (om tillämpligt)
- [ ] Länkar fungerar
- [ ] Språk är enkelt och tydligt
- [ ] Steg-för-steg utan hopp
- [ ] Troubleshooting för vanliga problem

## 🔄 Uppdateringar

När plattformen uppdateras, kontrollera:
- Versionnummer
- Nya dependencies
- Ändrade kommandon
- Nya features som bör nämnas
