5.1 Multi-Stage Pipeline

Mål:

Skapa en komplett CI/CD pipeline med flera stages

Pipeline Stages:

  1. Lint & Format: Black, flake8, pylint
  2. Unit Tests: Snabba tester
  3. Integration Tests: API tester
  4. Security Scan: Bandit, Safety
  5. UI Tests: Playwright E2E
  6. Performance Tests: Load testing
  7. Build: Docker image
  8. Deploy: Till staging/production

Uppdaterad Fil:

.github/workflows/ci-cd-pipeline.yml

jobs:
  lint:
    runs-on: ubuntu-latest
    steps:
      - run: black --check .
      - run: flake8 .

  unit-tests:
    needs: lint
    runs-on: ubuntu-latest
    steps:
      - run: pytest tests/unit/ -v

  api-tests:
    needs: unit-tests
    runs-on: ubuntu-latest
    steps:
      - run: pytest test_guestbook_api.py --html=report.html

  security-scan:
    needs: api-tests
    runs-on: ubuntu-latest
    steps:
      - run: bandit -r . -f json
      - run: safety check

  ui-tests:
    needs: api-tests
    runs-on: ubuntu-latest
    steps:
      - run: playwright install
      - run: pytest tests/ui/ -v
