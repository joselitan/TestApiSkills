🔐 Login Tests (5 tester)

Test Cases

  • test_login_page_loads - Verifiera att login sidan laddas
  • test_successful_login - Testa lyckad inloggning
  • test_failed_login_wrong_password - Testa fel lösenord
  • test_failed_login_empty_fields - Testa tomma fält

Användning

pytest tests/ui/test_login_ui.py -v

🔧 Fixtures

authenticated_page

Fixture som ger en redan inloggad sida. Sparar tid genom att inte behöva logga in i varje test.

def test_something(authenticated_page: Page):
    # Sidan är redan inloggad på /guestbook
    authenticated_page.fill("#name", "Test")

🚀 Hur Man Kör Testerna

Alla UI Tester

pytest tests/ui/ -v

Med Synlig Browser (Headed Mode)

pytest tests/ui/ --headed

Specifik Browser

pytest tests/ui/ --browser firefox
pytest tests/ui/ --browser webkit
pytest tests/ui/ --browser chromium

Slow Motion (För Debugging)

pytest tests/ui/ --headed --slowmo 1000

Med Screenshots

pytest tests/ui/ --screenshot only-on-failure

Med Video

pytest tests/ui/ --video retain-on-failure

🌐 Cross-Browser Testing

Stödda Browsers

  • ✅ Chromium (Chrome, Edge)
  • ✅ Firefox
  • ✅ WebKit (Safari)

Kör På Alla Browsers

pytest tests/ui/ --browser chromium --browser firefox --browser webkit

🐛 Debugging

Playwright Inspector

set PWDEBUG=1
pytest tests/ui/test_login_ui.py::test_successful_login

Pause Test

# I test koden:
page.pause()

⚠️ Viktigt att Veta

Förutsättningar

  • Flask app måste köra på http://localhost:8080
  • Playwright browsers måste vara installerade: playwright install
  • Databas måste vara initialiserad med test data
