✅ 1.1 Avancerad Test Rapportering

Implementerat

  • ✅ pytest-html för HTML test reports
  • ✅ pytest-cov för code coverage
  • ✅ Allure Framework med decorators
  • ✅ pytest.ini konfiguration

Användning

# HTML rapport
pytest tests/api/ --html=reports/test_report.html

# Coverage rapport
pytest tests/api/ --cov=. --cov-report=html

# Allure rapport
pytest tests/api/ --alluredir=reports/allure-results
allure serve reports/allure-results
