Problem: Databasen använder hårdkodad sökväg 'guestbook.db' som kan bete sig olika på Mac och Windows.
Lösning: Implementerade plattformsoberoende sökvägar med os.path.join()
# Tillagt:
import os

def get_db_path():
    basedir = os.path.abspath(os.path.dirname(__file__))
    return os.path.join(basedir, 'guestbook.db')

# Uppdaterat:
def init_db():
    conn = sqlite3.connect(get_db_path()) # Använder nu get_db_path()

def get_db():
    conn = sqlite3.connect(get_db_path()) # Använder nu get_db_path()
Fördelar:
  • Fungerar på både Mac (/) och Windows (\) filsystem
  • Databasen skapas alltid i projektets rotkatalog
  • Inga hårdkodade sökvägar
