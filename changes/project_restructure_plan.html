💻 Kod Ändringar

1. conftest.py

# FÖRE
from tests.factories import create_guestbook_entry, create_multiple_entries

# EFTER
from tests.fixtures.factories import create_guestbook_entry, create_multiple_entries

2. src/database.py

# FÖRE
DB_PATH = 'guestbook.db'
TEST_DB_PATH = 'guestbook_test.db'

# EFTER
DB_PATH = 'data/guestbook.db'
TEST_DB_PATH = 'data/guestbook_test.db'

3. templates/guestbook.html & login.html

# FÖRE
<link rel="stylesheet" href="{{ url_for('static', filename='style.css') }}">

# EFTER
<link rel="stylesheet" href="{{ url_for('static', filename='css/style.css') }}">

# Lägg till

✅ Fördelar med Ny Struktur

  • ✅ Tydlig separation - App code, tests, docs separerade
  • ✅ Lättare att hitta - Allt har sin plats
  • ✅ Bättre gitignore - Generated files i egna mappar
  • ✅ Skalbart - Lätt att lägga till nya features
  • ✅ Professionellt - Följer Python best practices
  • ✅ CI/CD vänligt - Tydliga paths för automation

⚠️ Risker & Mitigering

Risk Sannolikhet Mitigering
Imports slutar fungera Låg Testa efter varje ändring
Flask hittar inte templates Mycket Låg Flask hittar templates/ automatiskt
CSS path fel Låg Uppdatera HTML templates
Database path fel Låg Uppdatera database.py
GitHub Actions failar Låg Uppdatera paths i workflow

🔄 Rollback Plan

Om något går fel:

  1. Git commit innan vi börjar
  2. Om problem: git reset --hard HEAD
  3. Alla ändringar är dokumenterade här
  4. Kan manuellt flytta tillbaka filer
