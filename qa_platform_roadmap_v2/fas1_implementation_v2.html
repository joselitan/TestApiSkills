🛡️ 1.2 Rate Limiting

Beskrivning

Skydda API:erna mot överbelastning och DoS-attacker genom att begränsa antalet requests per användare/tid.

Implementation

# src/app.py
from flask_limiter import Limiter
from flask_limiter.util import get_remote_address

limiter = Limiter(
app=app,
key_func=get_remote_address,
default_limits=["200 per day", "50 per hour"]
)
@app.route('/api/login', methods=['POST'])
@limiter.limit("5 per minute")

# Login logic
pass

Fördelar

  • Förebygg DoS-attacker

  • Konfigurerbara limiter per endpoint
